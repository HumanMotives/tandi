# Jong Nederland Quiz MVP

Standalone HTML/CSS/JS demo voor GitHub Pages, Netlify of een vergelijkbare statische host.

## Aanpassingen in deze versie

- Kleuren aangepast naar de nieuwe Jong Nederland huisstijl:
  - Natuur groen: `#AACF64`
  - Bos groen: `#244522`
  - Zand beige: `#FAF2E8`
  - Zon geel: `#F5C368`
  - Water blauw: `#8FD4EC`
  - Nacht blauw: `#033B6E`
- Lettertypes aangepast naar de nieuwe huisstijl:
  - Koppen: Lilita One
  - Lopende tekst en interface: Montserrat
- De drie voordeelblokjes onder de intro zijn verwijderd.
- Extra organische/speelse route-elementen toegevoegd in de achtergrond.
- Bugfix: antwoordknoppen en CTA-knoppen blijven nu binnen de container, ook bij langere teksten of smallere schermen.

## Gebruik

Open `index.html` lokaal of upload de volledige map naar GitHub Pages/Netlify.

## Bestanden

- `index.html`: structuur van de module
- `styles.css`: styling en huisstijlkleuren
- `script.js`: quizvragen, scorelogica en resultaatweergave
